          Source links, model assumptions and deployment notes

          This is a client-side cockpit using public APIs directly. Wind and solar capacity inputs are GW fallback values used only when live NESO/Elexon capacity data cannot be read; they convert weather indices into approximate MW. The stress index is explanatory: it is not an official grid adequacy, security-of-supply or balancing-cost metric.

          • Carbon Intensity APINational/regional carbon intensity and generation mix.https://api.carbonintensity.org.uk/
          • Elexon Insights APIDemand outturn, actual generation, wind/solar forecasts, market prices and system prices.https://data.elexon.co.uk/bmrs/api/v1/
          • NESO Data PortalEmbedded wind/solar forecasts and embedded capacity fields in MW.https://api.neso.energy/api/3/action/
          • Open-Meteo Forecast APIGFS/UKMO/ECMWF weather variables for weather-derived wind and solar proxies.https://api.open-meteo.com/v1/forecast
